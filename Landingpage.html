Summary: Latitude vs. X

The objective for this project was to analyze how weather changes as you get closer to the equator. To accomplish this task, we used data from the OpenWeatherMap Api to gather a dataset on 500 Plus cities.

After gathering the dataset, we used Matplotlib to plot different components of the weather vs. latitude. We used for elements of the weather to determine the ideal climate. These elements of weather include temperature, cloudiness, windspeed and humidity. On this site you will find source data and visualizations we created with our analizations, explantions and trends /correlations that we found.
